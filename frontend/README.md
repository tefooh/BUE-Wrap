# BUEWrap Frontend

Standalone portfolio demo for BUEWrap.

This version is intentionally disconnected from any real backend. It runs with:
- Local test authentication
- Mock wrap data
- Client-side share link generation/resolution

## Demo login

- Username: `demo.student`
- Password: `demo1234`

## Run locally

```sh
npm install
npm run dev
```

The app starts on `http://localhost:8080` by default.

## Build

```sh
npm run build
```

## Stack

- Vite
- React + TypeScript
- Tailwind CSS
- Framer Motion
